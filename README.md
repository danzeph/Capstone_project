##  Inventory Management API

The **Inventory Management API** is a backend capstone project built using Django and Django REST Framework.

The system enables users to manage inventory items through secure authentication using JWT via Django REST Framework SimpleJWT.

###  Key Features

* User registration and authentication
* Inventory item CRUD operations
* Owner-based permission control
* Filtering, searching, and sorting inventory data
* Inventory change history tracking
* Pagination support

### Security

* JWT-based authentication
* User-specific inventory access
